# Valuation Rules

## Intrinsic Value Estimation

### Simplified DCF Method
1. Project normalized free cash flow for next 5 years using consensus or analyst estimates
2. Apply terminal growth rate of 2-3% (long-term GDP growth)
3. Discount at weighted average cost of capital (WACC)
4. Use conservative assumptions for growth and margins

### Relative Valuation Method
1. Identify 5-10 comparable companies in same industry
2. Calculate median PE, PB, PS, EV/EBITDA for the peer group
3. Adjust for growth differences (use PEG to normalize)
4. Apply peer median multiples to target company's fundamentals

### Asset Value Floor
1. Book value per share as minimum reference
2. Adjust for off-balance-sheet assets and liabilities
3. For distressed situations: liquidation value = assets minus liabilities at distressed sale prices (50-70% of book)

## Margin of Safety Requirements

| Certainty Level | Required Margin |
|----------------|----------------|
| High (stable business, predictable cash flows) | 30% minimum |
| Medium (some cyclicality, reasonable predictability) | 40% minimum |
| Low (turnaround, high cyclicality, emerging market) | 50% minimum |

Margin of safety = (Intrinsic Value - Current Price) / Intrinsic Value × 100%

A positive margin of safety means the stock is undervalued. Negative means overvalued.

## Valuation Level Classification

| Level | Margin of Safety | Interpretation |
|-------|-----------------|----------------|
| Deep Value | > 50% | Significant undervaluation, potential catalyst needed |
| Value | 30-50% | Undervalued, attractive entry point |
| Fair | -10% to 30% | Reasonably priced, neither cheap nor expensive |
| Expensive | -30% to -10% | Overvalued, limited upside |
| Very Expensive | < -30% | Significantly overvalued, high risk of correction |

## Decision Rules

1. **Strong Buy**: Deep value or value + favorable cycle score (0-4) + low risk
2. **Buy**: Value + neutral cycle score (5-6) + moderate risk
3. **Hold**: Fair value regardless of cycle position
4. **Reduce**: Expensive + elevated cycle score (7+) + high risk
5. **Sell**: Very expensive + extreme cycle score (9+) + very high risk

## Credit Spread Analysis (for bond/credit investments)

The credit spread must compensate for expected loss:
- Expected loss = probability of default × loss given default
- If spread > expected loss by a comfortable margin: attractive
- If spread < expected loss: inadequate compensation, avoid

Example: If historical default rate for B-rated bonds is 4% and recovery rate is 40%:
- Expected loss = 4% × (1 - 40%) = 2.4%
- If B-rated spread is 5%: adequate compensation (5% - 2.4% = 2.6% excess)
- If B-rated spread is 2%: inadequate compensation (2% - 2.4% = -0.4% shortfall)
